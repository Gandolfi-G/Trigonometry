01. Le cercle trigonométrique

Repère orthonormé, sens trigonométrique, rayon 1, points cardinaux et angles repères.

Points-clés

  • Cercle de centre O, rayon 1 : on parle de « cercle unité ».
  • Orientation sens trigonométrique (anti-horaire).
  • Angles repères : 0, π/2, π, 3π/2, 2π.
  • Points sur le cercle : (1,0), (0,1), (-1,0), (0,-1).

Exemples visuels

  • Placer le point M à 45° (π/4) : il est dans le 1er quadrant.
  • Placer le point N à 210° (7π/6) : il est dans le 3e quadrant.

Exercices rapides

  1. Donner les coordonnées exactes des points A, B, C, D du cercle unité.
  2. Dire dans quel quadrant se trouve un angle : 30°, 120°, 225°, 330°.
  3. Convertir : 360° → ? rad ; π → ? degrés.
